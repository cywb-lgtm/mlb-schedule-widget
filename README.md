# mlb-schedule-widget

A live MLB schedule widget for Mac built with Übersicht. Shows live scores, standings, streak, series info and upcoming schedule for all 30 teams.

## Requirements

- macOS
- [Übersicht](http://tracesof.net/uebersicht/)

## Installation

1. Download and install Übersicht from tracesof.net/uebersicht
2. On first launch, right-click → Open to bypass the macOS security warning
3. Download the latest release zip from this repo
4. Unzip and drag the `mlb-schedule.widget` folder into your Übersicht widgets folder (Übersicht menu bar → Open Widgets Folder)
5. The widget will appear on your desktop automatically

## Customization

To change the team, open `index.jsx` in a text editor and update the team name in the configuration at the top of the file.
